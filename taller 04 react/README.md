# Taller 04 Expo CRUD

Proyecto Expo en React Native para el taller de arquitectura de componentes visuales.

## Requisitos

- Node.js y npm
- Android Studio o un dispositivo Android con Expo Go o development build

## Ejecucion

1. Instala dependencias con `npm install`.
2. Inicia el proyecto con `npm run android`.

## Flujo

- Pantalla de listado con imagen, titulo y subtitulo.
- Pantalla de formulario para crear o actualizar elementos.
- Modal de confirmacion para eliminar y Toast nativo en Android.
